Detailed Description

A container is an element that contains other elements. It implements the required behaviour to navigate between children.

Note
It must be used when controller navigation is required between children! Using a standard Element will result in navigation being ignored.

It allows the selection to automatically move between elements without having to specify which ones are next to which.
